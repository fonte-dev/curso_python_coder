# Curso de Python: Coderhouse 🐍

Bienvenido a mi repositorio de aprendizaje. Aquí documento todo mi progreso, ejercicios y proyectos realizados durante el curso de Python en Coderhouse, construyendo los cimientos técnicos para mi desarrollo profesional como AI Engineer.

## 📌 Sobre este repositorio

Este espacio contiene el código fuente de los desafíos y prácticas vistos a lo largo de las 13 unidades del curso. El viaje de aprendizaje cubrió desde los fundamentos de la programación hasta el desarrollo web Full-Stack utilizando frameworks modernos.

## 🛠️ Tecnologías y Conceptos Aprendidos

A lo largo del curso, adquirí experiencia práctica en:
* **Fundamentos de Python:** Variables, bucles, condicionales y estructuras de datos.
* **Programación Orientada a Objetos (POO):** Clases, herencia, polimorfismo y encapsulamiento.
* **Manejo de Errores y Archivos:** Excepciones (try/except) y persistencia de datos.
* **Desarrollo Web con Django:** * Implementación del patrón de arquitectura MVT (Model-View-Template).
  * Creación y gestión de bases de datos relacionales (SQLite y Django ORM).
  * Operaciones CRUD completas (Create, Read, Update, Delete).
  * Vistas basadas en funciones y Vistas basadas en Clases (CBV).
  * Autenticación de usuarios, seguridad (decoradores) y gestión de sesiones.
* **Frontend Integrado:** Diseño responsivo utilizando HTML y Bootstrap 5.
* **Control de Versiones:** Uso de Git y GitHub para la gestión del código.

## 🚀 Proyecto Destacado: MERITUM

Como culminación de todo lo aprendido en el curso, desarrollé **MERITUM**, una aplicación web Full-Stack diseñada para documentar conocimientos, organizar ideas y compartir logros.
